# Project Instructions for Codex

## Project Purpose

This is a safe teaching project for learning Codex CLI.

The project is not a real research project. It is used to practice:

- reading code
- making small code edits
- running tests
- debugging errors
- checking git diff
- following project instructions

## Safety Rules

Do not modify files outside this project directory.

Do not delete files unless explicitly requested.

Before modifying code, explain:

1. what file will be changed
2. why the change is needed
3. how to verify the change

Prefer minimal changes.

Do not rewrite the whole project unless explicitly requested.

## Coding Style

Use Python 3.

Keep functions small and readable.

Add tests for new behavior.

Use Chinese when explaining to the user.

## Current Project Topic

This project implements basic object detection metrics:

- IoU
- TP / FP / FN
- Precision
- Recall
